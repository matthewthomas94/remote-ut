---
description: Default rules for the Designer Agent Kit
---

1.  **No Lorem Ipsum**: Use realistic content appropriate for the domain.
2.  **Semantic HTML & Accessibility**: Default to semantic tags (`<nav>`, `<main>`, `<button>`) and ensure ARIA attributes are used correctly.
3.  **React Functional Components**: Use modern React patterns (hooks, functional components). No class components.
4.  **Frequent Commits**: Commit often with descriptive messages. Do not commit directly to `main` for experimental changes.
5.  **Check Storybook First**: Before building a new component, check if an equivalent exists in Storybook.
6.  **Component States**: Handle `empty`, `loading`, `populated`, and `error` states for all data-driven components.
7.  **Browser Verify**: Always verify changes in a browser environment before presenting to the user.
